# نمونه داشبورد سرویس‌های دیجیتال همراه اول

- `index.html` → صفحه ورودی (ریدایرکت به داشبورد)
- `mci_real_dashboard.html` → داشبورد اصلی

1) Settings → Pages
2) Source: Deploy from a branch
3) Branch: `main` و Folder: `/ (root)`
4) Save
